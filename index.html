Rule Engine

Group by rule. Walk away.

Define URL patterns — contains, domain, glob, or full regex. Takt watches every new tab and files it into the right Chrome group with the right color. Manual or automatic. Free tier includes 5 rules with basic matching. Premium unlocks unlimited rules, all match types, title matching, and compound AND/OR conditions.

contains exact domain glob regex
Quick Switch

Find any tab. Instantly.

Press Alt+Shift+Space anywhere in Chrome. A fuzzy-search overlay appears — type a few letters and jump without touching your mouse. Searches all open tabs by title and URL. Premium extends results to hibernated tabs and your full browsing history.

fuzzy search hibernated tabs history search keyboard-only
Duplicate Prevention

No duplicates. No surprise.

Open the same URL twice and Takt closes the copy, focuses the original, and shows a quiet toast. Five matching strategies handle query strings, fragments, and trailing slashes. Bypass anytime with Shift+click or the context menu. Premium feature.

Resource Dashboard

See what your tabs cost.

Per-tab memory measurement. Per-group health scores from A to F. Network request counts. Staleness tracking for tabs you forgot about. Set memory budgets and Takt discards tabs that exceed them. All measured locally — no external services. Premium feature.

Hibernation

Sleep tabs you don't need.

Takt fully closes idle tabs and saves their state — URL, title, tab group, scroll position. Memory is reclaimed the moment the tab closes. Wake any tab with one click and it returns exactly where you left it. Manual hibernation is always free. Set idle thresholds, memory pressure limits, or a tab ceiling to hibernate automatically — Premium features.

Everything else.

The details that make the difference.

Profiles for every context

Create named rule sets — Work, Personal, Research. Switch in the popup or schedule auto-switching by time of day. Premium.

Keyboard-first

Alt+Shift+G to organize. Alt+Shift+U to ungroup. Alt+Shift+C to collapse. No mouse needed.

Rules you didn't write

Takt watches your unmatched tabs. When a domain cluster appears, it suggests a rule. Accept or dismiss. Free.

Right-click, done

Right-click any tab and choose "Create rule from this tab." Takt prefills the URL pattern and group name. Premium.

Portable settings

Export rules, settings, and profiles as a single JSON file. Import on another machine. Premium.

Save your workspace

Snapshot all your tab groups into a named session. Restore later in a new window or replace the current one. Up to 50 sessions. Premium.

Sync across devices

Pick a folder watched by your cloud client (Google Drive, Dropbox, iCloud) or connect Google Drive directly for automatic background sync. Premium.

Act on whole groups

Close, bookmark, or move an entire tab group to a new window in one click. Available from the Options page or right-click menu. Premium.
