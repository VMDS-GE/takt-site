Keyboard Shortcuts

Organize your tabs without touching the mouse.

Default shortcuts #

ShortcutActionAvailability
Alt + Shift + GOrganize all tabs in all windowsComplete

The keyboard shortcut requires a Complete subscription. Free users can use the popup buttons instead.

Customizing shortcuts #

Chrome lets you change extension keyboard shortcuts:

  1. Navigate to chrome://extensions/shortcuts
  2. Find Takt in the list
  3. Click the pencil icon next to "Organize all tabs"
  4. Press your preferred key combination

You can also clear the shortcut entirely if you prefer to use the popup exclusively.

Popup actions #

These actions are available from the popup (click the Takt toolbar icon):

  • Group All — organizes every tab across all windows that matches a rule
  • Ungroup All — removes all tab groups across all windows
  • Collapse All — collapses every tab group to save visual space

Each action applies to all open windows, not just the current one.
