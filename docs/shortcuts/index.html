Keyboard Shortcuts

Organize your tabs without touching the mouse.

Default shortcuts #

Shortcut Action
Alt + Shift + G Group all tabs across all windows
Alt + Shift + U Ungroup all tabs across all windows
Alt + Shift + C Collapse all tab groups across all windows
Alt + Shift + E Expand all tab groups across all windows
Alt + Shift + Space Quick-switch: search and jump to any open tab

Organize, ungroup, and collapse shortcuts are available to all users. Quick-switch (Alt + Shift + Space) requires Premium.

Quick-switch overlay #

The quick-switch overlay is a command-palette style search for your open tabs. Press the shortcut and start typing — tabs are filtered in real time by title and URL. Use ↑↓ to navigate, Enter to jump to the selected tab, Tab to autocomplete, and Esc to dismiss.

Premium expands the results to include hibernated tabs (shown with an ice-blue treatment and moon icon) and browsing history (shown in orange). Selecting a hibernated tab wakes it automatically. Selecting a history result opens that URL in a new tab.

On pages where content scripts cannot be injected (e.g. chrome:// pages), a popup window opens instead.

See the Quick Switch page for full documentation on keyboard interactions, result ranking, and edge cases.

Customizing shortcuts #

Chrome lets you change extension keyboard shortcuts:

  1. Navigate to chrome://extensions/shortcuts
  2. Find Takt in the list
  3. Click the pencil icon next to the shortcut you want to change
  4. Press your preferred key combination

You can also clear the shortcut entirely if you prefer to use the popup exclusively.

Popup actions #

The same actions are available from the popup (click the Takt toolbar icon):

Button Keyboard shortcut
Group All Alt + Shift + G
Ungroup All Alt + Shift + U
Collapse All Alt + Shift + C
Expand All Alt + Shift + E

Both the popup buttons and keyboard shortcuts apply to all open windows.
