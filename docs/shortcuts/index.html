Keyboard Shortcuts

Organize your tabs without touching the mouse.

Default shortcuts #

ShortcutAction
Alt + Shift + GGroup all tabs across all windows
Alt + Shift + UUngroup all tabs across all windows
Alt + Shift + CCollapse all tab groups across all windows
Alt + Shift + SpaceQuick-switch: search and jump to any open tab

All shortcuts are available to free and Premium users.

Quick-switch overlay #

The quick-switch overlay is a command-palette style search for your open tabs. Press the shortcut and start typing — tabs are filtered in real time by title and URL. Use ↑↓ to navigate, Enter to jump to the selected tab, Tab to autocomplete, and Esc to dismiss.

On pages where content scripts cannot be injected (e.g. chrome:// pages), a popup window opens instead.

Customizing shortcuts #

Chrome lets you change extension keyboard shortcuts:

  1. Navigate to chrome://extensions/shortcuts
  2. Find Takt in the list
  3. Click the pencil icon next to the shortcut you want to change
  4. Press your preferred key combination

You can also clear the shortcut entirely if you prefer to use the popup exclusively.

Popup actions #

The same actions are available from the popup (click the Takt toolbar icon):

ButtonKeyboard shortcut
Group AllAlt + Shift + G
Ungroup AllAlt + Shift + U
Collapse AllAlt + Shift + C

Both the popup buttons and keyboard shortcuts apply to all open windows.
