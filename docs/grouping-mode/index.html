Grouping Mode

Choose how Takt groups your open tabs — three exclusive modes to match your workflow.

Overview #

Grouping Mode determines how Takt organizes your tabs into tab groups. Each window uses exactly one mode at a time, selected from a radio group in the Takt popup. Your choice persists across sessions so you never have to re-select it.

The three available modes are: Rules (the default), Domain, and AI. Each mode is mutually exclusive — selecting one deactivates the others.

Modes #

Compare the three grouping modes side by side:

Mode What it does Tier
AI Uses Chrome's on-device Prompt API to invent semantic group labels from tab content Premium
Domain Groups tabs by their registrable domain so same-site tabs share a group Free
Rules Groups tabs using your configured rules — color, name, and conditions you define Free

Toggle #

The Takt popup shows a radio group with three options — Rules, Domain, and AI. Select the option you want, then click Group Tabs. Takt uses the selected mode for that grouping action.

Your choice persists across sessions (stored in chrome.storage.sync via saveSettings). Switching modes does not retroactively regroup already-open tabs — only the next Group Tabs action uses the newly selected mode.

Profiles & Schedules #

Profiles and Schedules only apply when Rules mode is active. If you switch to AI or Domain mode, Profiles and Schedules become inert and have no effect until you return to Rules mode.
