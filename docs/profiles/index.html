Profiles & Schedules

Save different rule sets as profiles and switch between them manually or on a schedule.

Premium feature. Profiles and scheduled switching require a Premium subscription.

What are profiles #

A profile is a named subset of your rules. When a profile is active, only the rules assigned to it are evaluated. Rules not in any profile are always active ("All Rules" mode).

Use profiles to separate different contexts:

  • Work — rules for Jira, GitHub, Confluence, Slack
  • Personal — rules for YouTube, Reddit, shopping sites
  • Research — rules for academic papers, documentation sites

Creating a profile #

Open the Options page and go to the Profiles tab:

  1. Click "Create Profile"
  2. Enter a name
  3. Choose to clone current rules (copies all rules into the new profile) or start empty
  4. Click Create

New rules you create while a profile is active are automatically added to that profile.

Switching profiles #

Click a profile card to activate it. Click "All Rules" to deactivate all profiles and use every rule.

When you switch profiles:

  • The rule engine reloads with only the active profile's rules
  • Existing tab groups are not removed — only new organization actions use the updated rules
  • Click "Group All" after switching to reorganize existing tabs

Scheduled switching #

Automate profile switching by time of day and day of week:

  1. On a profile card, click the schedule section
  2. Toggle "Enable schedule"
  3. Set a start time and end time
  4. Select which days of the week the schedule applies

When the schedule window begins, Takt activates the profile. When the window ends, it reverts to "All Rules" mode (unless you manually switched to a different profile).

Cross-midnight schedules are supported. A schedule from 10:00 PM to 6:00 AM activates at 10 PM and deactivates at 6 AM the next morning.

If multiple profiles have overlapping schedules, the first match wins (based on profile creation order).
