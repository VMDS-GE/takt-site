Group Actions

Act on entire tab groups at once — close, bookmark, or move them to a new window.

Premium feature. Group Actions require a Premium subscription.

$3/month $30/year save 17%

Close Group #

Closes all tabs in a group at once.

  • Pinned tabs are skipped. Pinned tabs in the group are left open; the count of skipped tabs is shown in the feedback toast.
  • Last-tab-in-window protection. If closing the group would leave a window completely empty (accounting for pinned tabs), Takt opens a blank new tab first so the window stays open.
  • The group is removed from Takt's registry after closing.

A confirmation prompt appears before closing, showing the number of tabs that will be closed.

Bookmark Group #

Saves all tabs in the group as Chrome bookmarks inside a Takt Bookmarks folder.

  • Root folder. Takt creates a single Takt Bookmarks folder in your Chrome bookmarks (under Other Bookmarks). If the folder already exists, it is reused.
  • Subfolder per save. Each bookmark action creates a timestamped subfolder if a subfolder with that group name already exists, so no bookmarks are overwritten.
  • Restricted URLs skipped. chrome://, chrome-extension://, and javascript: URLs cannot be bookmarked and are silently skipped.
  • Bookmarks permission required. The Bookmarks permission must be granted in Settings → Permissions before this action is available. If the permission is missing, a prompt guides you there.

The tabs themselves are not closed or moved — Bookmark Group is read-only.

Move Group to New Window #

Detaches all tabs in the group into a brand-new Chrome window.

  • Group properties preserved. The new window's group has the same title, color, and collapsed state as the original.
  • Cross-window consolidation. If the group is spread across multiple windows (consolidated), all tabs are gathered into the single new window.
  • Last-tab-in-window protection. Same as Close Group — if moving all group tabs would empty a source window, a blank tab is created first.
  • The registry is updated: the old group entries are removed and the new window's group is registered.

Accessing Group Actions #

Options page — Groups view

Open Options → Rules → Groups. Expand any group to see the Close, Bookmark, and Move to Window buttons. On a free plan, the buttons are visible but locked behind the Premium upgrade prompt.

Right-click context menu

Right-click any tab that belongs to a group. The Takt group actions submenu appears with three items corresponding to the three actions. If the tab is not in a group, a notification explains this. The submenu is only shown for Premium users.
