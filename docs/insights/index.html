Insights & Suggestions

Takt watches your browsing patterns and suggests rules for domains you visit frequently — so you can build a rule set without starting from scratch.

Overview #

The Insights pane in the popup surfaces auto-generated rule suggestions based on the domains you have open or visit often. Each suggestion card shows a domain that Takt has detected as a candidate for a new rule, along with a count of matching open tabs or history visits. Takt never stores your browsing history beyond the current session — analysis happens locally and on demand.

How it works #

Takt generates suggestions from two sources:

  1. Open tabs — Takt clusters your open tabs by domain. A domain must have at least 3 open tabs (MIN_CLUSTER_SIZE = 3) to appear as a tab-based suggestion.
  2. Browsing history — Takt scans your history for frequently-visited domains. A domain must meet either the minimum visit count or minimum active-days threshold to qualify. History-based results are capped at 20 per session.

Domains already covered by an existing rule are excluded from suggestions — Takt won't suggest a rule you already have.

Interactions #

Each insight card has two actions:

  • Create Rule — opens the Options page with the domain pre-filled in the rule editor, so you can create a rule with one click. The card is removed after the rule is saved.
  • Dismiss — permanently hides the suggestion. The card won't reappear until your browsing patterns change significantly enough for the domain to re-qualify. Dismiss is persistent across popup opens within the session.

Free vs Premium #

Premium Free users see up to 3 insight cards at a time. If Takt has found more suggestions than the free limit, an upgrade prompt appears below the visible cards inviting you to unlock the hidden ones.

Premium users see all suggestions with no cap. Upgrading removes the 3-card limit immediately — all pending suggestions become visible without reloading.

Edge cases and gotchas #

  • Existing rules are excluded. If a domain is already covered by an existing rule, it won't appear as a suggestion. This prevents duplicate rules.
  • Internal URLs are skipped. chrome:// and chrome-extension:// URLs are excluded from suggestion analysis — only regular web pages are considered.
  • No insights. If Takt has no active suggestions (all dismissed, all covered by rules, or not enough tabs), the Insights pane shows an empty state with a message. This is normal — add more tabs or visit more domains to generate new suggestions.
