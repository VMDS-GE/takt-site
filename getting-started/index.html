Getting Started

Install Takt, create your first rule, and organize your tabs in under a minute.

Install #

Add Takt from the Chrome Web Store. No account is required for free features.

After installing, click the puzzle piece icon in your toolbar and pin Takt for quick access. The extension icon shows a badge with the number of ungrouped tabs in your current window.

Takt requests only the permissions it needs. Optional permissions (for the resource dashboard and duplicate prevention) are requested when you enable those features. See Permissions Explained for details.

Create your first rule #

Open the extension options page by clicking the gear icon in the popup, or right-click the Takt icon and select "Options."

  1. Click "Add Rule" in the Rules tab.
  2. Name your rule — something descriptive like "GitHub" or "Work email."
  3. Add a condition — choose "URL" and "contains," then enter a pattern like github.com.
  4. Set the group name — this is the label Chrome shows on the tab group. For example, "Dev."
  5. Pick a color — free users can use grey and blue. Complete unlocks all 8 Chrome colors.
  6. Save — the rule is ready.

The "contains" match type checks if the pattern appears anywhere in the tab's URL. For more precise matching, see the Rule Reference.

Organize your tabs #

Click the Takt popup icon, then press "Group All." Every open tab that matches a rule will be moved into the correct tab group with the right name and color.

The popup also shows:

  • Which rule matches the currently active tab (if any)
  • Quick toggles to enable or disable individual rules
  • Ungroup All — removes all tab groups
  • Collapse All — collapses every tab group to save space

Enable auto-mode #

Complete feature. Auto-assign requires a Complete subscription.

With auto-mode enabled, Takt groups tabs automatically as they load — no clicking needed.

  1. Open the Options page and go to the Settings tab.
  2. Under Automation, select "Auto."
  3. New tabs matching a rule will be grouped instantly.
